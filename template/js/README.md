# Discord Bot Template Quick Guide

> **This README covers the Discord bot template itself.**
> For detailed instructions on installing and using the `create-discord-bot` CLI, see the separate CLI documentation.

---

## Purpose

This repository is a **ready-to-use Discord bot template** generated by the `npm create-discord-bot` tool. It provides a clean starting point for building feature-rich bots using [**Discord.js (v14+)**](https://discord.js.org/), with:

- Automatic slash command registration
- Dynamic event handling
- Configurable environment variables
- Built-in ESLint setup

Core patterns are based on the official [Discord.js guide](https://discordjs.guide) to handlers and project structure.

## Template Structure

```
<your‑bot‑folder>/          # Root of your generated bot
├── src/                    # All source code
│   ├── commands/           # Slash commands, organized by category folders
│   │   └── utility/        # e.g. ping.js, server.js, user.js
│   ├── events/             # Listener files (ready.js, messageCreate.js, interactionCreate.js)
│   ├── handlers/           # commandHandler.js & eventHandler.js
│   ├── config.js           # Loads values from .env
│   ├── deploy-commands.js  # Registers slash commands with Discord
│   └── index.js            # Bot entry file
├── .env.example            # Template for environment variables
├── eslint.config.js        # ESLint rules
├── package.json            # Dependency list and scripts
└── README.md               # This quick guide
```

## Setup & Usage

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Create your environment file**

   ```bash
    # Windows
    ren ".env.example" ".env"

    # macOS / Linux
    cp .env.example .env
   ```

   Fill in the values:

   ```env
   DISCORD_TOKEN=your-bot-token
   APP_ID=your-application-id
   GUILD_ID=your-server-id
   ```

3. **Register slash commands** (only when you add or update commands)

   ```bash
   npm run register
   ```

4. **Start the bot**

   ```bash
   npm start
   ```

Your bot will connect and log a confirmation in your console.

## Adding Commands & Events

### Commands

* Place a new file in `src/commands/<category>/`.
* Export an object with `data` (a `SlashCommandBuilder`) and `execute(interaction)`.
* Example:

  ```js
  // src/commands/fun/joke.js
  import { SlashCommandBuilder } from 'discord.js';

  export default {
    data: new SlashCommandBuilder()
      .setName('joke')
      .setDescription('Tells a random joke'),
    async execute(interaction) {
      await interaction.reply('Why did the bot cross the road?...');
    },
  };
  ```
* Run `npm run register` to sync with Discord, then `npm start` to load it.

### Events

* Add or modify files in `src/events/`, each exporting:

  ```js
  export default {
    name: 'ready',
    once: true,
    async execute(client) {
      console.log(`Logged in as ${client.user.tag}!`);
    },
  };
  ```
* The handler auto-registers all event files when the bot starts.

## Configuration

All settings are loaded from `.env` via `src/config.js`. You can extend this file with more environment variables (e.g., `PREFIX`, `OWNER_ID`).
